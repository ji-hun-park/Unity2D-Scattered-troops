# Unity2D-Scattered-troops
## 플레이 방법
- 마우스 드래그 앤 드롭: 유닛 선택
- 마우스 우클릭: 선택된 유닛 이동
- A: 공격 모드로 전환(다시 A를 눌러 일반 모드로 전환하거나, 우클릭으로 공격 대상 지정 가능)
- S: 선택된 유닛들 좁은 범위로 산개시키기
- D: 선택된 유닛들 넓은 범위로 산개시키기
- ESC: 메뉴 열기

## 게임 소개
유닛들을 <span style="color:red">**산개**</span>시켜 적의 공격을 피하는 게임
